[WizBox Readme + Help]

	[Program Not Launching/Opening?]

 First off, if your program doesn't open up, you're most likely missing the .NET 3.1 framework
 which can be found at https://dotnet.microsoft.com/en-us/download/dotnet/3.1 or by opening
 the shortcut in this folder.
 
	[Config Concerns]
 
 (If you are having issues with the program, and have .NET 3.1 framework, then you should
 delete your config file and WizBox will create a new one. You will have to reselect your
 WizardGraphicalClient.exe in the settings.)
 
 The config is all auto generated, and the program handles it all. You shouldn't ever edit this, especially while the program is running because it 
 has a good chance to mess something up.
